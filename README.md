# Novel Reader

A novel reader application with FastAPI backend and Vue 3 frontend.

## Features

- FastAPI backend with async support
- Vue 3 frontend with modern UI
- Docker deployment ready
- Comprehensive test coverage

## Quick Start

```bash
docker-compose up
```

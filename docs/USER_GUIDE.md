## 快速开始

### 系统要求
- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **屏幕分辨率**: 推荐 1280x720 或更高
- **网络**: 本地网络访问或互联网连接（用于爬虫功能）

### 首次访问
1. 打开浏览器，访问 `http://localhost`
2. 首次使用需要注册账户
3. 登录后即可开始使用

## 用户注册与登录

### 注册新账户
1. 点击登录页面的"注册新账户"链接
2. 填写用户名（3-20个字符，仅字母、数字、下划线）
3. 设置密码（至少8位，包含字母和数字）
4. 点击"注册"按钮

### 登录
1. 输入用户名和密码
2. 点击"登录"
3. 登录成功后将自动跳转到书库首页

### 密码安全提示
- 使用强密码，包含大小写字母、数字和特殊字符
- 定期更换密码
- 不要在公共设备上勾选"记住我"

## 书库首页

### 浏览书籍
- 首页以网格卡片形式展示所有书籍
- 每张卡片显示：彩色侧边条、书名、作者、章节数
- 点击卡片进入书籍详情页

### 搜索书籍
- 在顶部搜索框输入关键词
- 支持按书名、作者搜索
- 实时显示搜索建议

### 书籍操作
- **阅读**: 点击书籍卡片进入阅读器
- **收藏**: 点击卡片上的星形图标
- **删除**: 管理员可删除书籍（谨慎操作）

### 扫描本地书籍
- 点击"扫描书籍"按钮
- 系统将扫描配置的本地目录
- 新发现的书籍将自动添加到书库

## 阅读器

### 基本阅读
- 点击章节标题进入阅读
- 使用键盘方向键或屏幕按钮翻页
- 滚动鼠标滚轮浏览内容

### 阅读设置
点击右下角的设置按钮可调整：

| 设置项 | 说明 |
|--------|------|
| 主题模式 | 浅色/深色/跟随系统 |
| 字体大小 | 12px - 32px 滑动调节 |
| 行高 | 紧凑/标准/宽松 |
| 字体 | 系统默认/宋体/黑体 |

### 目录导航
- 点击左侧目录按钮打开章节列表
- 点击章节标题快速跳转
- 当前阅读章节高亮显示

### 阅读进度
- 系统自动保存阅读位置
- 下次打开自动恢复到上次位置
- 支持跨设备同步（需登录）

### 上下章切换
- 阅读到底部自动显示"下一章"按钮
- 顶部显示"上一章"按钮
- 键盘快捷键：← 上一章，→ 下一章

## 搜索功能

### 书籍搜索
1. 在首页搜索框输入关键词
2. 按 Enter 或点击搜索按钮
3. 查看搜索结果列表

### 内容搜索（全文搜索）
1. 进入搜索页面
2. 选择"内容搜索"标签
3. 输入关键词，可限定在特定书籍内搜索
4. 查看包含关键词的章节片段

### 搜索建议
- 输入时自动显示建议
- 建议包括：书名、作者、标签
- 点击建议直接跳转

## 收藏夹

### 添加收藏
- 在书库首页点击书籍卡片的星形图标
- 或在书籍详情页点击"收藏"按钮

### 管理收藏
- 进入"收藏夹"页面
- 查看所有收藏的书籍
- 点击书籍继续阅读
- 点击取消收藏按钮移除

### 收藏同步
- 登录状态下收藏自动同步到服务器
- 换设备登录后可恢复收藏列表

## 爬虫下载

### 添加下载任务
1. 进入"爬虫"页面
2. 粘贴小说目录页 URL
3. 点击"开始下载"
4. 等待任务完成

### 任务状态
- **等待中**: 任务已创建，等待执行
- **下载中**: 正在抓取章节
- **已完成**: 所有章节下载成功
- **失败**: 下载过程中出错

### 注意事项
- 请遵守目标网站的 robots.txt 规则
- 合理设置并发数，避免对目标网站造成压力
- 部分网站可能需要特殊处理

## 常见问题

### Q: 忘记密码怎么办？
A: 目前版本不支持密码找回，请联系管理员重置密码。

### Q: 书籍显示乱码？
A: 请确保书籍文件为 UTF-8 编码。如为 GBK 编码，请使用工具转换。

### Q: 爬虫下载失败？
A: 检查以下几点：
- URL 是否正确
- 目标网站是否可访问
- 是否被目标网站封禁
- 尝试降低并发数

### Q: 如何备份数据？
A: 备份以下目录：
- `./data/novel.db` - 数据库文件
- `./data/books/` - 书籍文件
- `./data/index/` - 搜索索引

### Q: 支持哪些书籍格式？
A: 目前支持纯文本 (.txt) 格式。每章一个文件，文件名格式：`章节序号_章节标题.txt`

### Q: 如何导入本地书籍？
A: 将书籍文件放入 `./data/books/` 目录，然后在管理后台点击"扫描书籍"。
